# 📝 Заполнение полей Port Forwarding

## 🔧 Правило 1: Бэкенд (API)

**Порт сервиса:** `8000-8000` или просто `8000`

**IP-адрес:** `192.168.0.102`

**Внутренний порт:** `8000` (или оставьте пустым)

**Протокол:** `TCP` (или `ВСЕ` если TCP нет в списке)

**Состояние:** `Включено`

**Общий порт сервиса:** Оставьте `---Выберите---` (не заполняйте, так как порт 8000 не входит в стандартные протоколы)

---

## 📋 Краткая инструкция

1. **Создайте первое правило для бэкенда:**
   - Порт сервиса: `8000`
   - IP-адрес: `192.168.0.102`
   - Внутренний порт: `8000` (или пусто)
   - Протокол: `TCP` или `ВСЕ`
   - Состояние: `Включено`

2. **Создайте второе правило для фронтенда:**
   - Порт сервиса: `5173`
   - IP-адрес: `192.168.0.102`
   - Внутренний порт: `5173` (или пусто)
   - Протокол: `TCP` или `ВСЕ`
   - Состояние: `Включено`

3. **Сохраните настройки**

4. **Перезагрузите роутер** (если требуется)

## ✅ После настройки

Запустите проект:
```bash
docker-compose -f docker-compose.production.yml up -d
```
